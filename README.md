# squint-vite-plugin

A Squint plugin for [Vite](https://vitejs.dev/). Very early in it's development.

## Demos
### React + Squint + Vite
This is the vite + react template with squint added. To play around with it:
```sh
cd demo/react
# install dependencies
npm i
# run dev mode
npm run dev
# run build
npm run build
# run preview
npm run preview
```
