# squint-vite-plugin

A Squint plugin for [Vite](https://vitejs.dev/). Very early in it's development.

To play around with the demo, you can do the following:

```sh
cd demo/
# dev mode
npm run dev
# build and preview
npm run build
npm run preview
```
